# 🚀 mlops-journey

Мой путь в MLOps с нуля. Первый проект — предсказание оттока клиентов.

## 🎯 Цель
Научиться строить воспроизводимый ML-пайплайн: от данных до продакшн-модели.

## 🛠 Технологии
- Python 3.11
- scikit-learn
- (позже) Docker, MLflow, DVC, Airflow

## 🚀 Быстрый старт
```bash
make setup     # Установить зависимости
make train     # Обучить модель
make predict   # Сделать предсказание
make test      # Запустить тесты
